MaazDB Ecosystem & Drivers

Connect to MaazDB seamlessly using our official, high-performance client libraries. Each driver is tailored specifically for its ecosystem, providing a secure, idiomatic, and lightning-fast way to interact with your database.

MaazDB-JS

A high-performance, asynchronous Node.js client library. It leverages native Promises and implements the custom MaazDB binary protocol securely over TLS 1.3.

GitHub npm

Installation

npm install maazdb-js

Quick Start

const MaazDB = require('maazdb-js');

async function main() {
  const db = new MaazDB();
  try {
    await db.connect("127.0.0.1", 8888, "admin", "admin");
    console.log("✓ Connected to MaazDB");

    await db.query("CREATE DATABASE web_app;");
    await db.query("USE web_app;");
    
    const results = await db.query("SELECT * FROM users;");
    console.log(results);
  } finally {
    db.close();
  }
}
main();

MaazDB-RS

A memory-safe, zero-cost abstraction Rust SDK. Implements the binary protocol over a secure TLS 1.3 socket, supporting both blocking and tokio-based workflows.

GitHub Crates.io

Installation (Cargo.toml)

[dependencies]
maazdb-rs = "0.1.0"

Quick Start

use maazdb_rs::{MaazDB, Result};

fn main() -> Result<()> {
    let mut db = MaazDB::connect(
        "127.0.0.1", 8888, "admin", "admin"
    )?;
    println!("✓ Connected via TLS 1.3");

    db.query("USE store_prod;")?;
    
    let res = db.query("SELECT 'Hello!' as msg;")?;
    println!("{}", res);

    Ok(())
}

MaazDB-Py

A pure-Python client library. No heavy C-extensions required. Features automatic TLS 1.3 encryption and Context Manager support for safe connection handling.

GitHub PyPI

Installation

pip install maazdb-py

Quick Start

from maazdb import MaazDB

try:
    # Context manager handles closing
    with MaazDB() as db:
        db.connect("127.0.0.1", 8888, 
                   "admin", "admin")
        print("✓ Connected to MaazDB")
        
        db.query("USE analytics;")
        res = db.query("SELECT COUNT(*) FROM users;")
        print(f"Total users: {res}")
except Exception as e:
    print(f"Error: {e}")
