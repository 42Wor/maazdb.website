MaazDB Ecosystem & Drivers

MaazDB is more than a database engine. Build applications faster with our native, high-performance client libraries that handle encryption, binary protocols, and connection pooling automatically.

MaazDB-JS

A high-performance, asynchronous Node.js client library. It leverages native Promises and Node.js Buffers to implement the custom MaazDB binary protocol securely over TLS 1.3. Perfect for real-time web applications.

GitHub npm

Installation

npm install maazdb-js

Usage

const MaazDB = require('maazdb-js');

async function main() {
    const db = new MaazDB();
    // Connect securely via TLS 1.3
    await db.connect("127.0.0.1", 8888, "admin", "admin");
    
    const results = await db.query("SELECT * FROM users;");
    console.log(results);
}

MaazDB-RS

The official high-speed Rust SDK built on Tokio. It implements the binary protocol over a secure TLS 1.3 socket, allowing Rust applications to communicate with your database safely with zero-cost abstractions.

GitHub Crates.io

Cargo.toml

maazdb-rs = "0.1.0"

Usage

use maazdb_rs::{MaazDB, Result};

fn main() -> Result<()> {
    let mut db = MaazDB::connect("127.0.0.1", 8888, "admin", "admin")?;
    
    db.query("CREATE DATABASE store_prod;")?;
    let res = db.query("SELECT * FROM users;")?;
    Ok(())
}

MaazDB-Py

A pure-Python client library with automatic TLS 1.3 handshake. It supports context managers (with statements) for safe connection handling and provides a simple, idiomatic Pythonic API.

GitHub PyPI

Installation

pip install maazdb-py

Usage

from maazdb import MaazDB

# Supports Context Managers
with MaazDB() as db:
    db.connect("127.0.0.1", 8888, "admin", "pass")
    
    db.query("INSERT INTO logs VALUES ('System Start');")
    print(db.query("SELECT * FROM logs;"))
