MaazDB Ecosystem & Drivers

MaazDB is more than a database engine. Build applications faster with our native, high-performance client libraries that handle encryption and protocols automatically.

MaazDB-JS

Async-first Node.js driver featuring zero-copy parsing and native Promise support. Perfect for real-time web applications and APIs.

GitHub npm

Installation

npm install maazdb-js

Usage

const db = new MaazDB();
await db.connect("127.0.0.1", 8888, "root", "pass");

MaazDB-RS

High-speed Rust SDK built on Tokio. Provides type-safe query building and strict binary protocol alignment for maximum efficiency.

GitHub Crates.io

Cargo.toml

maazdb-rs = "0.1.0"

Usage

let db = MaazDB::connect("addr", 8888, "user", "pass").await?;

MaazDB-Py

A pure-Python client library with automatic TLS 1.3 handshake. Fully supports context managers and provides a simple, idiomatic Pythonic API.

GitHub PyPI

Installation

pip install maazdb-py

Usage

with MaazDB() as db:
    db.connect("127.0.0.1", 8888, "user", "pass")
