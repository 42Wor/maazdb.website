MaazDB v11.6 Documentation

MaazDB v11.6 is an ultra-high-performance, columnar, disk-based relational database engine written in Rust. It utilizes LSM Trees (Log-Structured Merge-Trees) for fast writes, Columnar Storage for analytical reads, and XOR Encryption for data security.

1. Installation & Configuration

Configuration File (`maazdb.toml`)

The configuration file is generated automatically on first launch if it does not exist.

  • Linux: /etc/maazdb/maazdb.toml
  • Windows: C:\Program Files\MaazDB\maazdb.toml
[server]
host = "0.0.0.0"
port = 8888

[storage]
data_dir = "data"

2. Using the Command Line Interface

Start MaazDB server and connect with the CLI client:

# Start server
maazdb-server

# In another terminal, connect
maazdb-cli

3. Authentication

Security is mandatory. You must login before running queries.

LOGIN admin admin;

Manage Users

CREATE USER maaz WITH PASSWORD 'secure';
DROP USER maaz;

4. Database Management

MaazDB is a multi-tenant system. You must Create and Use a database.

CREATE DATABASE inventory_system;
USE inventory_system;
BACKUP snapshot_v1; -- Creates a backup folder

5. Data Types

MaazDB uses Strict Static Typing.

SERIALInt32Auto-increment Primary Key
TEXTUTF-8String values
INTInt32Whole numbers
DOUBLEFloat64Decimal numbers (Prices)
BOOLBytetrue/false
TIMESTAMPISO-8601Date and Time

6. DDL - Create Table

CREATE TABLE products (
    pid SERIAL PRIMARY,
    title TEXT,
    price DOUBLE,
    in_stock BOOL
);

DROP TABLE products;

7. INSERT Data

Add new rows to a table. You can specify columns explicitly.

-- Explicit Insert
INSERT INTO products (title, price, in_stock) VALUES ('Gaming Mouse', 49.99, true);

-- Auto-Defaults (ID generates automatically)
INSERT INTO products (title, price) VALUES ('Keyboard', 89.99);

8. SELECT Data

Retrieve data with powerful filtering and sorting.

Basic Select

SELECT products;

Filtering (WHERE)

-- Find items cheaper than $50
SELECT products WHERE price < 50.0;

-- Find items that are in stock AND expensive
SELECT products WHERE in_stock = true AND price > 100.0;

Sorting (ORDER BY)

-- Sort by Price (High to Low)
SELECT products ORDER BY price DESC;

-- Sort by Title (A-Z)
SELECT products ORDER BY title ASC;

Limiting Results

-- Get top 5 most expensive items
SELECT products ORDER BY price DESC LIMIT 5;

9. UPDATE Data

Modify existing records based on a condition.

-- Update price for a specific item
UPDATE products SET price = 45.00 WHERE title = 'Gaming Mouse';

-- Mark all items as out of stock (Dangerous!)
UPDATE products SET in_stock = false WHERE price < 10.0;

10. DELETE Data

Remove records permanently.

-- Delete a specific item by ID
DELETE FROM products WHERE pid = 5;

-- Delete all out of stock items
DELETE FROM products WHERE in_stock = false;
