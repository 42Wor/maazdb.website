Getting Started

1. Install & Launch

Run the installer for your OS. MaazDB automatically sets itself up as a **background service** (Windows/Linux/macOS) and is ready for use immediately.

No Rust or dependencies required!

2. Connect with CLI

Open your terminal (CMD, PowerShell, Bash) and connect to the server using the bundled command-line client.

maazdb-cli

The default host is `127.0.0.1:8888` (configurable in `maazdb.toml`).

3. Authenticate & Use

Log in, create a database, and switch to it before running any queries.

LOGIN admin admin;
CREATE DATABASE sales;
USE sales;

Key Features

LSM-Tree + Columnar
High write throughput (LSM) combined with highly optimized analytical read speeds (Columnar). The best of both worlds.
Crash-Safe & Self-Healing
Full WAL (Write Ahead Log) ensures **Zero Data Loss**. The server automatically replays the log and restarts if it crashes.
Strict SQL & Indices
Robust support for complex SQL commands including `WHERE`, `ORDER BY`, `LIMIT`, and auto-handling of Primary and Secondary Indices.
Cross-Platform Daemon
Installed as a native system service/daemon on all major platforms for reliable, continuous background operation.
